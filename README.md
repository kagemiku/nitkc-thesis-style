# nitkc-thesis-style
Style file of LaTeX for graduation thesis at NIT, Kagoshima College.

# Style files
## abstract
* none

## thesis
* **thesis/nitkc-thesis.sty**

### Usage
#### `\cover` command
Creation cover page.
```tex
\begin{document}
\cover{Title}{Student No}{Name of you}{Name of supervisor}{Filing date(day only)}
.
.
.
\end{document}
```

#### `\toc` command
Creation table of contents.
```tex
\begin{document}
.
\toc
.
.
.
\end{document}
```
